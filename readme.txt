This is a single player shooting game.

Keyboard Controls:
Up arrow -> Increase power
Down arrow -> Decrease power
a -> Rotate anticlockwise
c -> Rotate clockwise
spacebar -> Shoot the top
1 -> Player View
2 -> Top View
3 -> Overhead view
4 -> Helicopter view
5 -> Follow view
Esc -> Quit

Mouse Controls:
In view 4 (i.e Helicopter view) scroll to set the viewing distance.

Game includes extra features like puasing the game by pressing 'p' and again play it by pressing 'p'.
